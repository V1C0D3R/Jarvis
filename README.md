# Jarvis

Tested on Mac OS X 10.11 (El Capitan)

## Requirements
- Python 2/3 with _myo-python_ and _phue_
- Myo Connect installed
- A brain because Jarvis can not do everything for you

## Usage

From your folder, in Terminal, begin to play with :
```
$ python jarvis.py
```
